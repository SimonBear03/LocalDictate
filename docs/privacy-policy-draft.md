# LocalDictate Privacy Policy Draft

LocalDictate is designed as a local-first macOS dictation app.

## Data Processing

LocalDictate records audio only when the user starts dictation. The audio is used to produce a transcript on the user's Mac through Apple speech recognition when supported.

## Data Collection

LocalDictate V1 does not collect, sell, share, or transmit user data. The app does not include third-party analytics, advertising SDKs, or account tracking.

## Local Storage

Dictation history, cleanup text, settings, and diagnostics are stored locally on the user's Mac. The user can manage local app data from within the app and through macOS.

## Permissions

LocalDictate may request:

- Microphone access to record dictation.
- Speech Recognition access to transcribe speech.
- Accessibility access to paste the user's dictated text into the active text field.

Accessibility access is used only for user-initiated text insertion.
